# Bayes error

O erro de Bayes é simplesmente um erro irredutível de previsão que está associada a uma atividade específica. É o erro mínimo que o modelo consegue atingir para a aplicação em que está sendo utilizado Normalmente é relacionado com um erro humano, próximo de 0%.
